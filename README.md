# form-validator
form validator that checks user inputs and displays error messages 

## Features
- Validates form inputs using JavaScript
- Displays error and success messages
- Prevents form submission with invalid data
- Simple and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. User enters data into the form fields.
2. JavaScript checks the input values.
3. Error messages are shown for invalid inputs.
4. The form is submitted only when all fields are valid.
